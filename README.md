# Math Dash

Math Dash is a browser-based math runner game built with HTML, CSS, and JavaScript.

## How to play
- Press **Space** to start the game.
- Press **Space** again to jump over obstacles.
- Answer math questions to clear obstacles and earn points.
- A green timer bar shows how much time remains to answer the current question.
- At score 1000, a boss fight begins.
  - Dodge 10 math symbols.
  - Then complete 6 brick-throw questions to defeat the boss and move to Level 2.

## Files
- `index.html` — game page structure
- `styles.css` — visual styling
- `main.js` — game logic and interaction

## Publish on GitHub Pages
1. Create a GitHub repository and push these files to it.
2. In repository settings, enable **GitHub Pages**.
3. Choose the branch (`main` or `master`) and folder (`/root` or `/docs`) that contains `index.html`.
4. Visit the provided GitHub Pages URL to play.
